# Open Source Software Audit Project (24BCE10382)

## Project Overview
This project involves an extensive audit of an open-source software system, which requires both theoretical knowledge and implementation skills in Linux shell scripting.  
The objective of this project is to learn about the philosophy, functions, and application of open-source software.

---

## About Mozilla Firefox
Mozilla Firefox is a web browser developed by Mozilla. It is developed with the aim to maintain the privacy and transparency of the users. Mozilla Firefox is different from all the other browsers in the sense that the users can view the source code and can contribute to it as well.

---

## Project Objectives
- Understand the philosophy behind open-source  
- Analyze the Firefox browser as an example of an open-source system  
- Perform system-level operations using Linux shell scripting 

---

## Project Structure

script1.sh - Displays system information (OS, kernel, user, uptime) 
script2.sh - Checks whether Firefox is installed 
script3.sh - Audits disk usage and directory permissions 
script4.sh - Analyzes log files for specific keywords 
script5.sh - Generates an open-source manifesto 

---

## How to Run the Project

### Navigate to Project Folder

bash
cd ~/your-folder-name

### Create Script Files
touch script1.sh script2.sh script3.sh script4.sh script5.sh

### Make Scripts Executable
chmod +x *.sh

### Run Scripts
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
